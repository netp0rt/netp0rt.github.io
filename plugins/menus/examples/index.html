右键菜单插件

一个轻量级、可定制的右键菜单解决方案

下载插件 查看文档

主要特性

易于定制

通过JSON或XML就可定制右键菜单

示例展示

示例一

Code Menus配置部分：

                                
/* JSON with Commponents */
"menus": {
    // ...
    "mainMenu": [
        {
            "name": "mainMenu Example 1",
            "id": "main.example",
            "icon": "@fa:fa-solid fa-1",
            "action": "this.tst.s('MainMenu 1 toast test')"
        },
        {
            "name": "mainMenu Example 1",
            "id": "main.example",
            "icon": "@fa:fa-solid fa-1",
            "submenus": [
                {
                    "name": "submenu Example 1",
                    "id": "main.submenu.example1",
                    "icon": "@fa:fa-solid fa-1",
                    "action": "this.tst.s('Submenu Example 1 toast test')"
                }
            ]
        }
    ]
    // ...
},
"triggers": {
    ".test1": "mainMenu"
    // ...
}
